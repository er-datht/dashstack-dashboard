## MODIFIED Requirements

### Requirement: Message list header
The message list view SHALL display a top bar with a select-all checkbox on the far left, a search input (rounded, placeholder "Search") to its right, and grouped action buttons (download, info, trash) with segmented borders on the right.

#### Scenario: Top bar renders with select-all checkbox
- **WHEN** the message list view is displayed
- **THEN** the top bar shows a select-all checkbox, then the search input, then the action buttons
